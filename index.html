🤖 MaycolAIUltra-MD

Esta es una herramienta para configurar y ejecutar el bot MaycolAIUltra-MD para WhatsApp con tema de Hanako Kun. El bot se ejecuta en la consola, no en esta página web.

Instrucciones:

  1. Ejecuta node setup.js para instalar el bot (si aún no lo has hecho)
  2. Edita Goku-Black-Bot-MD/config.js para personalizar tu bot
  3. Ejecuta node start.js para iniciar el bot
  4. El bot se conectará automáticamente usando el número 51921826291
